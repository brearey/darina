# QWEN.md — Проект «Электронный фотоальбом Дарины Хоютановой»

## Обзор проекта

Это статический одностраничный веб-сайт — электронный фотоальбом, разделённый на три категории:
- **Деловой стиль** — 3 фотографии
- **Краски** — 2 фотографии
- **Личное** — 1 фотография

Изображения отображаются в каруселях Bootstrap, переключаемых через вкладки навигации.

## Структура проекта

```
├── index.html        — Главная страница (навигация + три карусели)
├── style.css         — Пользовательские стили для каруселей
├── script.js         — Динамическое заполнение каруселей изображениями
├── img/              — Папка с фотографиями
├── README.md         — Заголовок проекта
├── QWEN.md           — Этот файл
└── .git/             — Git-репозиторий
```

## Запуск проекта

Проект не требует сборки или зависимостей. Достаточно открыть `index.html` в браузере:

```bash
# Вариант 1: просто открыть файл
xdg-open index.html        # Linux
open index.html            # macOS
start index.html           # Windows

# Вариант 2: через локальный сервер (рекомендуется)
python3 -m http.server 8080
# Затем открыть http://localhost:8080
```

## Соглашения по коду

- **JavaScript:** vanilla JS без фреймворков, `DOMContentLoaded` для инициализации.
- **CSS:** минимальные кастомные стили поверх Bootstrap. Ограничение высоты изображений — `80vh` с `object-fit: contain`.
- **Именование файлов изображений:** кириллические названия с пробелами (например, `деловой стиль 1.jpg`).
- **Язык страницы:** русский (`lang="ru"`).

## Добавление новых фотографий

Чтобы добавить фото в существующий альбом, отредактируйте массив `images` в `script.js`:

```js
albums.business.images.push('img/новое фото.jpg');
```

Для новой категории — добавьте новый объект в `albums`, соответствующую вкладку в `index.html` и секцию `tab-pane`.
